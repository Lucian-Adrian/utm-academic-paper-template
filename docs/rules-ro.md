# Reguli si formate UTM pentru rezumat, articol si teza/proiect

Document consolidat din:

- `Abstract_template-Conf-UTM-2026-rom.docx`;
- `Paper_template-Conf-SMD-UTM-2026-rom.docx`;
- `sustinereTezeLicenta.pdf`;
- verificarea folderelor vechi `UTM Latex Template` si `Thesis_Report_Template_EN`.

Scopul documentului este sa ofere o referinta practica pentru redactare, verificare si recreare in LaTeX a unui articol academic UTM. Pentru articole se aplica in primul rand modelul `Paper_template-Conf-SMD-UTM-2026-rom.docx`; regulile de teza/proiect se folosesc numai pentru documente de licenta sau pentru pagini optionale cerute separat.

## Regula de prioritate

Daca regulile difera, se alege tipul documentului inainte de redactare:

- rezumat de conferinta: se aplica `Abstract_template-Conf-UTM-2026-rom.docx`;
- articol/lucrare de conferinta: se aplica `Paper_template-Conf-SMD-UTM-2026-rom.docx`;
- teza/proiect de licenta: se aplica `sustinereTezeLicenta.pdf`;
- cerinta mai stricta de la profesor, coordonator, catedra sau comisie: se aplica cerinta mai stricta si se noteaza abaterea fata de model.

Nu se combina limita de 12 000 caractere pentru articol cu limita de 60-80 pagini pentru teza.

## Regula profesorului pentru enumerari

La enumerari:

- fraza introductiva se termina cu doua puncte;
- fiecare element incepe cu litera mica;
- fiecare element se termina cu punct si virgula;
- ultima intrare pastreaza punctul si virgula daca profesorul a cerut explicit uniformitate;
- regula nu se aplica listei bibliografice numerotate, unde intrarile bibliografice se termina conform stilului de referinte.

Exemplu corect:

```text
Contributiile principale ale lucrarii sunt:
a) elaborarea modelului de analiza;
b) validarea rezultatelor obtinute;
c) formularea recomandarilor finale;
```

## Reguli academice generale

- lucrarea trebuie sa fie originala;
- toate sursele pentru idei, texte, figuri, tabele, scheme, formule, imagini scanate sau materiale adaptate trebuie citate;
- plagiatul include si preluarea figurilor, tabelelor sau schemelor fara citare;
- autorul raspunde pentru exactitatea si veridicitatea rezultatelor;
- titlul trebuie sa fie scurt, precis, fara acronime neconsacrate si inteligibil pentru nespecialisti;
- prescurtarile se explica la prima aparitie;
- simbolurile si notatiile se explica la prima aparitie, daca nu exista lista de notatii;
- stilul trebuie sa fie impersonal, fara persoana I;
- limbajul trebuie sa fie clar, direct, logic si coerent;
- in romana se folosesc diacritice;
- se evita formularile vagi, ambigue si afirmatiile nesustinute.

## Setari comune pentru modelele de conferinta

Pentru rezumat si articol:

- format hartie: A4;
- orientare: portret;
- margine sus: 2.0 cm;
- margine jos: 2.0 cm;
- margine dreapta: 2.0 cm;
- margine stanga: 2.5 cm;
- distanta antet: 1.0 cm;
- distanta subsol: 1.25 cm;
- font principal: Times New Roman;
- text principal: 12 pt;
- aliniere text principal: justificata.

## Rezumat de conferinta

Ordinea elementelor:

1. titlul lucrarii;
2. autorii;
3. afilierile;
4. autorul corespondent;
5. indrumatorul/coordonatorul stiintific;
6. rezumatul;
7. cuvintele cheie.

Titlul:

- se scrie cu majuscule;
- se centreaza;
- se scrie cu Times New Roman, 14 pt;
- se scrie cu caractere aldine;
- se foloseste spatiere la un rand.

Autorii:

- format: `Prenumele NUMELE`;
- se centreaza;
- Times New Roman, 12 pt;
- aldine;
- fiecare autor are indicator numeric pentru afiliere;
- autorul corespondent se marcheaza cu `*`.

Afilierile:

- includ departament, grupa, facultate, universitate, oras, tara;
- se centreaza;
- 10 pt;
- italic;
- fiecare afiliere incepe cu indicatorul numeric corespunzator.

Autorul corespondent:

- incepe cu `*Autorul corespondent:`;
- include prenume, nume si e-mail;
- se centreaza;
- 10 pt;
- stil normal.

Indrumatorul/coordonatorul stiintific:

- include numele, titlul stiintific si afilierea;
- se centreaza;
- 10 pt;
- eticheta si numele sunt aldine in model.

Rezumatul:

- incepe cu `Rezumat.`;
- eticheta este bold italic;
- corpul este italic;
- Times New Roman, 12 pt;
- aliniere justificata;
- lungime aproximativa: jumatate de pagina;
- include scopul cercetarii, metodologia, rezultatele principale si concluziile relevante;
- evidentiaza noutatea lucrarii;
- evita generalitatile, ambiguitatea si prescurtarile neexplicate.

Cuvintele cheie:

- incep cu `Cuvinte cheie:`;
- eticheta este bold italic;
- corpul este italic;
- Times New Roman, 12 pt;
- aliniere justificata;
- intre 4 si 6 cuvinte cheie;
- litere mici;
- separate prin virgule;
- evita repetarea termenilor din titlu.

## Articol/lucrare de conferinta

Structura normala:

1. titlu;
2. autori;
3. afilieri;
4. autor corespondent;
5. indrumator/coordonator stiintific;
6. rezumat;
7. cuvinte cheie;
8. introducere si formularea problemei;
9. parte experimentala sau metodologica;
10. rezultate si discutii;
11. concluzii;
12. multumiri, daca este cazul;
13. referinte;
14. declaratie de originalitate si asumare a responsabilitatii.

Limite:

- articolul are maximum 12 000 caractere cu spatii;
- rezumatul are 150-200 cuvinte;
- in rezumat nu se introduc citari bibliografice.

Textul principal:

- Times New Roman;
- 12 pt;
- justificat;
- paragrafele noi trebuie sa fie vizibile;
- se foloseste numerotarea automata a paginilor.

Sectiunile:

- titlurile din model sunt Times New Roman, 12 pt, aldine;
- se folosesc titluri scurte si paralele;
- sectiuni tipice: `Introducere`, parte metodologica/experimentala, `Rezultate si discutii`, `Concluzii`, `Multumiri`, `Referinte`.

Cerinte stiintifice:

- lucrarile originale si comunicarile scurte trebuie sa contina constatari originale;
- detaliile trebuie sa permita reproducerea experimentelor;
- articolele de sinteza trebuie sa ofere informatie noua si unificare de idei, nu doar rezumate ale lucrarilor anterioare.

Tabele:

- se centreaza;
- trebuie sa ocupe toata latimea paginii, cand este posibil;
- daca se continua pe pagina urmatoare, se marcheaza continuarea si se repeta rubricile;
- textul din tabel: 10 pt, spatiere la un rand;
- antetele coloanelor sunt explicative, aldine si includ unitati de masura unde este cazul;
- se numeroteaza in ordinea aparitiei: `Tabelul 1`, `Tabelul 2`;
- se citeaza in text ca `Tab. 1`, `Tab. 2`;
- titlul tabelului apare deasupra tabelului;
- titlul tabelului este 11 pt si aldine.

Figuri:

- se centreaza;
- se plaseaza cat mai aproape de prima mentiune in text;
- se numeroteaza in ordinea aparitiei: `Figura 1`, `Figura 2`;
- se citeaza in text ca `Fig. 1`, `Fig. 2`;
- au legenda scurta si exacta;
- legenda incepe cu `Figura` si numarul, cu caractere aldine;
- legenda se centreaza;
- legenda are 11 pt;
- imaginile trebuie sa aiba rezolutie buna, fara fisiere disproportionat de mari.

Ecuatii si formule:

- se foloseste Equation Editor, MathType sau echivalent LaTeX corect;
- ecuatiile se centreaza;
- se numeroteaza consecutiv cu cifre arabe in paranteze rotunde;
- numarul se plaseaza la marginea dreapta;
- in text se citeaza ca `Ec. (1)`, `Ec. (2)`.

Simboluri, notatii si unitati:

- simbolurile se scriu italic, 12 pt;
- vectorii se scriu aldine;
- functiile standard, precum `log`, `ln`, `exp`, se scriu cu font normal;
- se folosesc simboluri internationale acceptate;
- se recomanda unitatile SI;
- pentru litru se foloseste `L` mare;
- pentru kilogram se foloseste `k`, nu `K`;
- numerele zecimale se scriu cu punct, nu cu virgula;
- formulele si schemele mari se prezinta ca figuri.

Referinte:

- in text se folosesc numere arabe in paranteze patrate;
- referintele se numeroteaza in ordinea citarii;
- citarea se plaseaza inaintea semnului de punctuatie;
- forme acceptate: `[1]`, `[1-4]`, `[1,3]`;
- toate referintele citate trebuie sa existe in lista si invers;
- numele complet al revistei se scrie integral, fara abrevieri;
- lista de referinte este Times New Roman, 12 pt;
- se recomanda Zotero, EndNote, Mendeley sau ReferenceManager.

Modele de referinte:

```text
1. AUTOR 1, A.B.; AUTOR 2, C.D. Titlul lucrarii. Numele revistei Anul, Volumul, pp. 154-196, doi:
2. AUTOR 1, A.; AUTOR 2, B. Titlul cartii, 3rd ed.; Editura: Locatia editurii, Tara, An; pp. 154-196.
3. AUTOR 1, A.B. Titlul tezei. Nivelul tezei, Universitatea, Localitatea, Data sustinerii.
4. Titlul site-ului. Disponibil online: URL (accesat la data de Ziua Luna Anul).
```

Declaratia:

- se include la final pentru articolul din model;
- autorul corespondent declara originalitatea lucrarii;
- se confirma citarea corecta a tuturor surselor;
- se include nume/prenume, semnatura si data.

## Teza/proiect de licenta

Volum si proportii:

- memoriu explicativ: 60-80 pagini fara anexe;
- partea teoretica: aproximativ 30-40% si nu peste 40%;
- partea aplicativa/practica: aproximativ 60-70%;
- figurile si graficele nu depasesc 30% din volum.

Structura:

1. coperta;
2. pagina de titlu;
3. sarcina/caietul de sarcini;
4. pagina de dedicatii si multumiri, optional;
5. declaratia studentului;
6. avizul conducatorului;
7. rezumatul;
8. cuprinsul;
9. lista tabelelor, optional;
10. lista figurilor/graficelor/diagramelor/schemelor, optional;
11. lista abrevierilor, optional;
12. introducerea;
13. continutul pe capitole;
14. concluziile;
15. bibliografia;
16. anexele, daca este cazul.

Redactare:

- limba acceptata: romana, engleza, franceza sau rusa;
- A4;
- tiparire doar pe fata foii;
- Times New Roman, 12 pt;
- spatiere 1.5 randuri;
- titlurile capitolelor si paragrafelor pot fi 13-14 pt si aldine;
- lucrarea se organizeaza pe capitole si paragrafe, cu exceptia introducerii si concluziilor.

Rezumatul tezei:

- prezinta ideile principale, rezultatele si concluziile;
- include numarul de capitole, pagini, figuri, tabele si surse bibliografice;
- varianta romana are 300-500 cuvinte;
- varianta in engleza sau franceza traduce versiunea romana, inclusiv titlul.

Bibliografia tezei:

- contine toate sursele utilizate;
- are 15-30 titluri;
- poate include carti, monografii, lucrari de conferinta, articole, teze, rapoarte, standarde, norme, ghiduri, pagini web si rapoarte interne;
- citarea se poate face numeric in ordinea aparitiei sau alfabetic dupa primul element;
- citarea unei pagini se poate face ca `[8, p. 231]`;
- anul se scrie cu cifre arabe;
- daca sunt mai mult de trei autori, se poate folosi `et. al.` dupa autorii indicati.

Sustinerea:

- durata prezentarii: pana la 10-15 minute;
- se prezinta tema, scopul, sarcinile, variantele de solutionare, rezultatele, argumentarea, calculele tehnico-economice, masurile de securitate/protectie si concluziile;
- afirmatiile importante trebuie sustinute prin date, grafice, diagrame, tabele, scheme, formule, simulari sau rezultate software;
- raspunsurile la intrebari trebuie sa fie scurte, clare si directe;
- cu o saptamana inainte se recomanda mai multe sustineri preliminare.

## LaTeX recomandat pentru articol

Template-ul modern creat in acest repo este:

- `utm-academic-paper-template/utmconf.cls`;
- `utm-academic-paper-template/main.tex`;
- `utm-academic-paper-template/tools/check_utm_article.py`;
- `utm-academic-paper-template/titlepage-demo.tex`, doar pentru pagina de titlu UTM optionala.

Reguli LaTeX:

- pentru articolul de conferinta se foloseste `utmconf.cls`;
- logo-ul UTM este inclus in `assets`, dar nu este activ implicit deoarece modelul oficial de articol nu cere logo;
- daca profesorul cere antet cu logo, se activeaza `\UTMShowLogotrue`;
- pentru diacritice se compileaza cu Tectonic/XeTeX si Times New Roman;
- rezumatul se scrie in mediul `rezumat`;
- cuvintele cheie se scriu cu `\cuvintecheie{...}`;
- tabelele se eticheteaza cu `\utmtablecaption{...}`;
- figurile se eticheteaza cu `\utmfigurecaption{...}`;
- enumerarile de continut folosesc mediul `utmenumerate` si fiecare `\item` se termina cu punct si virgula.

Comenzi de verificare:

```powershell
cd "D:\uni\year2\guidelines\paper\utm-academic-paper-template"
python tools\check_utm_article.py
tectonic main.tex
tectonic titlepage-demo.tex
```

## Checklist final

- tipul documentului este ales corect;
- structura corespunde tipului documentului;
- formatul paginii si marginile sunt corecte;
- fontul, dimensiunea si alinierea sunt corecte;
- rezumatul respecta lungimea ceruta;
- cuvintele cheie sunt 4-6, cu litere mici, separate prin virgule;
- enumerarile respecta regula profesorului;
- tabelele, figurile si ecuatiile sunt numerotate si citate in text;
- referintele sunt citate in text si apar in lista;
- nu exista citari in rezumatul articolului;
- toate materialele preluate sunt citate;
- textul romanesc are diacritice;
- declaratia este inclusa cand modelul o cere.
